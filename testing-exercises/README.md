# Testing Exercises Project

This is a safe sandbox for learning and practicing tests without touching your main ToDo app.

## Commands

```bash
# Install deps (run once)
npm install

# Run tests in watch mode
npm test

# Run tests once
npm run test:run
```

## What's Inside

- `src/js/utils.js` - Functions to test
- `tests/utils.test.js` - Tests covering exercises 1-5
- `vitest.config.js` - Vitest + jsdom setup

Happy testing! 🎯
